# GoalSync v2.1

Your intelligent daily planner and progress tracker.  

---

## 🚀 Features
- Tasks, Goals, Projects (with participants, milestones, subtasks).
- Dashboard with charts + insights.
- Smart Assistant (Firestore-powered).
- Settings (profile, dark mode, notifications, integrations).
- Clean Apple-style design, responsive, dark mode.

---

## 🛠️ Setup
1. Open `js/firebase-config.js`.
2. Replace placeholders with your Firebase config.
3. Serve locally:
   ```bash
   firebase serve
